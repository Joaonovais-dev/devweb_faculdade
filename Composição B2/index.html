Por aqui vou mostrar oque aprendemos durante o semestre

Durante o semestre aprendemos a criar páginas web usando HTML e CSS. Aprendemos a estruturar o conteúdo usando tags HTML e a estilizar as páginas usando CSS. Também aprendemos sobre design responsivo, que é a prática de criar páginas que se adaptam a diferentes tamanhos de tela, como em dispositivos móveis e desktops. Além disso, exploramos conceitos de acessibilidade para garantir que nossas páginas sejam utilizáveis por pessoas com diferentes habilidades. No geral, foi um semestre cheio de aprendizado e desenvolvimento de habilidades em web design.
